## For Beginner Course

**Dev 2DGame with Python**

To run a python game using Arcade library you need to install 2 thing:

- [Python](https://www.python.org/) running enviroment
- [Arcade](https://arcade.academy/) library
- [VSCode](https://code.visualstudio.com/download) editor

To install Python enviroment in Windows, check out [HERE](https://arcade.academy/installation_windows.html).
to install Python enviroment in Mac, check out [HERE](https://arcade.academy/installation_mac.html)

After you successfully install the Python enviroment, now it time to install Arcade library.

Open command window(Terminal in Mac), then execute this:

```
pip3 install arcade
```

Last, download VSCode as your python program editor, and pick up a python plugin, you would be ready as a professional to start your coding journey.

**Dev 2DGame with Lua**

Using Love2D to develop 2d game is much more intuitive than using python arcade, you just need to [DOWNLOAD](https://love2d.org/) a love2d program as an interpreter of you game code.

<Image src="downloadlove2d.png" colStart="null|2" w="full" overflow="hidden"/>

After that, you still need an editor [VSCode](https://code.visualstudio.com/download) to write lua code. Last but not the least, there is a very handy plugin [Love2d Made Easy](https://marketplace.visualstudio.com/items?itemName=lwz7512.love2d-made-easy) to facilitate the lua writing and love2d program running process, saving your lots of time.
