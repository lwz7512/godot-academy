## Solutions

2DGame engines in Python language could be named just a few:

- [Pygame](https://www.pygame.org/wiki/GettingStarted), the most popular python game engine.
- [pyglet](http://pyglet.org/), windowing and multimedia library for Windows, Mac OS X and Linux.
- [Arcade](https://arcade.academy/), simillar to Pygame but more easier and engine-like rather than a primitive library.
