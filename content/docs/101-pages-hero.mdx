---
title: Add a hero
excerpt: Let's start by adding a hero to the page.
---

We are going to use [components](/docs/components) to build the hero.

```md title=content/pages/about.mdx
---
title: Welcome to Reflex
excerpt: Description for this page. This is mostly used for the description meta tag.
---

// highlight-start

<Section>
  <Container>
    <H1>Welcome to my site</H1>
    <P>This is the a short description for the page</P>
    <Image src="placeholder.jpg" />
  </Container>
</Section>

// highlight-end
```

This will render a hero as shown below:

<Div borderWidth="1px">
  <Span d="block" p="4" color="borderHover">
    Preview
  </Span>
  <Section>
    <Container>
      <H1>Welcome to my site</H1>
      <P>This is the a short description for the page</P>
      <Image src="placeholder.jpg" />
    </Container>
  </Section>
</Div>

<Callout my="8" pt="0" pb="2">

Place the _placeholder.jpg_ image file inside the `/content/images` directory.

</Callout>
