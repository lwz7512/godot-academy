---
title: Presets
excerpt: Reflex includes a few presets you can use as starting point for your themes.
---

You can see presets in action in the [Blocks library](/library).

<Image src="reflex-blocks-library-screenshot.jpg" />

## Base

```sh
npm i @reflexjs/preset-base typeface-inter
```

### Extend base

```js title=src/@reflexjs/gatsby-theme-base/theme.js
import typeface-inter
import base from "@reflexjs/preset-base"

export default {
  preset: base,
  // Overrides.
  colors: {
    primary: `purple`,
  },
}
```

## Flex

```sh
npm i @reflexjs/preset-flex typeface-open-sans typeface-prata
```

### Extend flex

```js title=src/@reflexjs/gatsby-theme-base/theme.js
import typeface-open-sans
import typeface-prata
import flex from "@reflexjs/preset-flex"

export default {
  preset: flex,
  // Overrides.
  colors: {
    primary: `#09f`,
  },
}
```

You can also use any [Theme UI presets](https://theme-ui.com/packages/presets).
