## Videos landing page

To customize the `/videos` landing page, create `src/@reflexjs/gatsby-theme-video/videos.js`:

View [source](https://github.com/reflexjs/reflex/blob/master/packages/gatsby-theme-video/src/videos.js).

```jsx title=src/@reflexjs/gatsby-theme-video/videos.js
import * as React from "react"
import { Article, H2, P, Div, Grid, Span } from "@reflexjs/components"
import { Link } from "@reflexjs/gatsby-theme-core"
import { Image } from "@reflexjs/gatsby-plugin-image"

export const Videos = ({ videos }) => {
  // Add custom video display here.
}
```

See more examples in the [blocks library](/library/blocks#videos).

View [source](https://github.com/reflexjs/reflex/blob/master/packages/gatsby-theme-video/src/video.js).

```jsx title=src/@reflexjs/gatsby-theme-video/video.js
}
```

## Add header and footer

To add a header and footer to the video pages, create the following blocks.

```
site
└── content
    └── blocks
        └── videos-header.mdx <-- /videos header // highlight-line
        └── videos-footer.mdx <-- /videos footer  // highlight-line
        └── video-header.mdx <-- video page header // highlight-line
        └── video-footer.mdx <-- video page footer // highlight-line
    ├── images
    ├── navs
    ├── pages
    └── videos
```

The blocks are placed on the corresponding pages automatically.
