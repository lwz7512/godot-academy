Use the `@reflexjs/gatsby-theme-video` package to add a video section to your site.

## Installation

**Step 1: add dependencies**

Add the `@reflexjs/gatsby-theme-video` package.

```sh
npm i @reflexjs/gatsby-theme-video
```

**Step 2: enable the plugin**

Edit your `gatsby-config.js` file and add the plugin.

```js title=gatsby-config.js
require(`dotenv`).config()

module.exports = {
  siteMetadata: {
    title: "Reflex",
    description: "Starter for Reflex.",
    siteUrl: process.env.SITE_URL || "http://localhost:8000",
  },
  plugins: [
    "@reflexjs/gatsby-theme-base"
    "@reflexjs/gatsby-theme-video" // highlight-line
  ],
}
```

**Step 3: run your site**

Start the development server.

```sh
gatsby develop
```

This will add the following directories to your site.

```
site
└── content
    └── blocks
    ├── images
    ├── navs
    ├── pages
    └── videos
        └── . <-- Place videos here. // highlight-line
```
